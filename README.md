# workout-note
A simple android APP to record workout.

# Basic Features

- Manage sessions
- Manage exercises in session
- Manage sets in exercise with weight and repetitions  
- Built-in Stopwatch
- Rest notification (30 seconds to 2 minutes)

# Tech
- UI: Jetpack Compose
- Database: Room
- Dependency Injection: Hilt

# Publish
- Google Play
